Aqui ^^ podemos ver que ocultamos el b, sin embargo si vamos a la linea de js que tiene el textcontent, podremos ver su contenido

La escala de importancia de las etiquetas para mostrar informacion de la clase elegida son:

.textContent Muestra el texto sin estilos aplicados

.innerHTML Muestra el texto con estilos aplicados Y el código HTML sin ser interpretado por consola, o interpretado con los estilos aplicados

.outerHTML Muestra el texto con estilos aplicados y el codigo HTML tanto del contenido y cada uno de sus childs, como del propio container class que se seleccionó
